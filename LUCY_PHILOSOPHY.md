## TL;DR

Lucy is **not a chatty AI**. She's a calm **navigator** who says relevant things at the right time - not a constant stream of info.

---

## How to Use Lucy

### Start her
```bash
lucy-start
# Lucy: "I'm here. What do you need?"
```

### Ask questions
```bash
lucy-ask "How do I optimize this code?"
# Lucy explains naturally
```

### Check file info
```bash
lucy-read-changes src/main.py
# Lucy: "main.py. 145 lines."
```

### Adjust her pace
```bash
lucy-set-speed 0.9
```
