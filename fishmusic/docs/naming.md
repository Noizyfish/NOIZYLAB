# Naming Conventions

## Sessions

```
FM_<PROJECT>_<YYYY-MM-DD>_S<##>
```

| Component | Format | Example |
|-----------|--------|---------|
| Prefix | `FM_` | Fish Music |
| Project | UPPERCASE | `GORUNFREE` |
| Date | ISO 8601 | `2024-12-04` |
| Session | `S##` | `S01`, `S02` |

## Audio Files

```
<PROJECT>_<ELEMENT>_<VERSION>.<ext>
```

| Element | Description |
|---------|-------------|
| `VOX` | Vocals |
| `BEAT` | Drums/percussion |
| `BASS` | Bass elements |
| `SYNTH` | Synthesizers |
| `FX` | Sound effects |
| `MIX` | Mixdowns |
| `MASTER` | Final masters |

## Versions

- `v1`, `v2`, `v3` - Major iterations
- `v1a`, `v1b` - Minor variations
- `v1_FINAL` - Approved version

## Examples

```
GORUNFREE_VOX_v2.wav
GORUNFREE_BEAT_v1a.wav
GORUNFREE_MIX_v3_FINAL.wav
```
