# NoizyLab – Client SOP (v1)

## 1) Create Client Workspace
- Run: NoizyLab – New Client (Quick Action)
- Verify: Client folder + Intake / Report / Invoice exist.

## 2) Intake Call / Session
- Fill in Intake.md (live while talking).
- Confirm:
  - Main use (work / creative / family)
  - Pain points
  - Backup status
  - Urgency / deadlines.

## 3) Triage Pass (Before Deep Work)
- Quick health:
  - Disk space
  - CPU / RAM behavior
  - Startup items
  - SMART / drive health (if possible)
- Note findings in Report.md under "What I Found."

## 4) Calm Tech Work (Deep Clean & Optimize)
- Remove junk / obvious bloat.
- Tune startup / background apps.
- Run OS + core app updates (if safe).
- Enable / verify backups.
- Note everything in "What I Did."

## 5) Summarize in Plain English
In Report.md → "Summary":
- 2–3 sentences, human language:
  - What was wrong
  - What you did
  - How it's better now.

## 6) VIP Care Plan Offer
- At the end, explain calmly:
  - "We can keep this from sliding back into chaos with a regular checkup plan."
- **Always ask:** "Do you want me to send you the VIP Care outline so this doesn't drift back into chaos again?"
- Options:
  - Monthly / Quarterly check
  - Priority support.

## 7) Invoice & Follow-Up
- Fill in Invoice.md.
- Send report + invoice.
- 3–5 days later: short check-in message & ask for a testimonial.
