## Setup Instructions

1. Open **System Settings**
2. Go to **Accessibility → Voice Control**
3. Enable Voice Control
4. Click **Commands → Custom**
5. Click **+** to add each command
6. Enter the voice phrase
7. Select "Keyboard Shortcut" as action
8. Record the keyboard shortcut

## Tips for Quadriplegic Users

- Use "wake up" and "go to sleep" to control listening
- Say "show commands" to see available commands
- "Show numbers" overlays clickable elements
- "Click [number]" to activate elements
- Combine with Switch Control for more options

## Alternative: Dictation Commands

For text input without keyboard:
- Enable Dictation: System Settings → Keyboard → Dictation
- Use `Fn Fn` (double-tap) to activate
- Say "new line" for Enter
- Say "cap" before words to capitalize
