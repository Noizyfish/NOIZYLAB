## Installation

1. Copy the folder structure to your Kontakt Quick Load directory:
   - **macOS**: `~/Library/Application Support/Native Instruments/Kontakt/Quick Load/`
   - **Windows**: `%APPDATA%\Native Instruments\Kontakt\Quick Load\`

2. Add your NKI files to the appropriate categories

3. Restart Kontakt to see the updated Quick Load browser

## 8Dio Libraries Organization

For 8Dio libraries (Hybrid Tools, Century Series, etc.):

1. Open Kontakt
2. Go to **Files** tab
3. Navigate to **Quick Load**
4. Drag your 8Dio NKI files into the appropriate categories

## Orchestral Tools (SINE Player)

Note: Orchestral Tools uses the SINE player, not Kontakt. However, older libraries may be in Kontakt format.

## Tips

- Use consistent naming conventions
- Group by library first, then by instrument type
- Create shortcuts for frequently used patches
- Regularly backup your Quick Load structure
